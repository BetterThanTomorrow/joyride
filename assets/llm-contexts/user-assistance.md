# This guide has moved

The Joyride extension now bundles comprehensive skills directly. If you are
reading this because a `copilot-instructions.md` file told you to fetch it,
the instructions in that file are outdated. The Joyride extension's bundled
skills contain everything that was here and more.

If these skills are available to you (check your skill list), load them
instead of relying on this file. If you don't have access to bundled skills,
the sources are at:

- `assets/skills/joyride/SKILL.md` — core Joyride skill (eval tools, async
  patterns, VS Code API, Flares, JS interop, available libraries)
- `assets/skills/joyride-user-scripting/SKILL.md` — user-scope scripting
  (activation scripts, keyboard shortcuts, disposables, npm deps)
- `assets/skills/joyride-workspace-scripting/SKILL.md` — workspace-scope
  scripting (project-local scripts, team sharing, classpath precedence)
